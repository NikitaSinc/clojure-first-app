# clojure-first-app

Project for learn how to develop clojure full-stack web application.

Backend libraries:
- http-kit
- seancorfield/next.jdbc
- healthsamurai/dsql

PS: router and middleware are written by me for learning puposes.

Frontend libraries:
- thheller/shadow-cljs
- reagent
- re-frame
- heathsamurai/macrocss

Database: postgresql
